Privacy Policy

Last updated: March 28, 2026

The short version

Traktor does not collect, store, transmit, or share any data. Period.

What Traktor does

Traktor is a local desktop application that extracts WordPress .wpress backup files on your computer. It reads a file from your disk, extracts its contents to a folder on your disk, and that's it.

Data collection

Traktor collects no data whatsoever. Specifically:

  • No analytics or telemetry. We do not track how you use the app.
  • No network requests. Traktor never connects to the internet. It works entirely offline.
  • No accounts or registration. There is no sign-up, login, or user profile.
  • No crash reports. If the app crashes, nothing is sent anywhere.
  • No cookies or tracking. The app has no web components.

Files you process

The .wpress files you open with Traktor are read locally on your machine. The extracted contents are written to your local disk. No file data ever leaves your computer.

Settings

Traktor stores minimal preferences locally on your machine (last-used directory, window position) using your operating system's standard settings storage (macOS Preferences, Windows Registry, Linux config files). These never leave your device.

This website

This website (traktor.wp-migration.com) is hosted on GitHub Pages. GitHub may collect standard web server logs (IP addresses, browser type) as described in GitHub's Privacy Statement. We do not add any additional tracking, analytics, or cookies to this website.

Open source

Traktor is open source under the GPLv3 license. You can inspect the source code to verify every claim on this page.
