Traktor is ready to use

Open Traktor from your Applications folder or Launchpad. On first launch, you will be prompted to set up AI agent integrations.
Command-line tool is available in your terminal:

traktor list backup.wpress
traktor cat backup.wpress wp-config.php
traktor extract backup.wpress ./output/
AI agent integration — register with your coding agents:

traktor mcp register

For help, run traktor --help or visit github.com/servmask/Qtraktor
